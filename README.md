# Weather Forecast App

## Introduction

This is a weather forecast app that uses the WeatherAPI to fetch weather data based on the user's location. The app displays the weather data in Fahrenheit or Celsius and changes the look of the page based on the data.

## Features

- **Location-Based Weather**: At the loading of the window, the weather is displayed based on the IP address.
- **Current Time Weather**: The app displays the current time weather data.
- **Search**: Users can search for a specific location to get the weather data for that location. and search suggestions are available.
